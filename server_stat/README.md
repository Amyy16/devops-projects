# SERVER PERFORMANCE STATS

## Overview
Bash script that collects basic Linux server performance statistics. 
It is useful for system monitoring, debugging, and learning DevOps practices.
This script provides information about CPU, memory, disk usage, processes, and logged-in users, and can optionally include uptime and failed login attempts.

## Features
- Total CPU usage 
- Memory usage
- Disk usage
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- Server uptime and load average
- Number of logged-in users

## Getting Started
### 1.  Clone the Repository
```
  git clone https://github.com/Amyy16/devops-projects.git
  cd server_stat

```
### 2. Make the Script executable
```
  chmod +x server_stat.sh

```

### 3. Execute the script

```
  ./server_stat.sh
```
